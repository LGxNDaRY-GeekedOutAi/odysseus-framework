Adaptive Stage Execution Internals

Stage startsengine sets stage_id, label, pattern, capabilities
-->
Parallel specialistsnon-critic personas run concurrently on staged_goal
-->
Critic synthesiscritic merges outputs, risks, and corrections
-->
Checkpoint + artifactstage status and artifact persisted to session state
-->
Optional asset queuemedia jobs queued by mode/stage capability rules

If any stage times out or fails, the workflow stops with checkpoint retention for recovery. If all stages pass, the run emits completion.
